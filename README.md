# todoList
This is an ongoing project that it, at this time, is still only just a to-do list. You can: 
1. Add tasks to a list 
2. Cycle between all/active/completed tasks 
3. Delete tasks 
4. Drag tasks to different spots in the list. 
New features such as note-taking and being able to create more than one list, will be added in the future.

How to use the program:
1. Clone the repository from github.
2. Open in Visual Studio Code.
3. Navigate to index.html
4. Right click on the code, scroll down and select "Open with Live Server"
5. Start making you own list and have fun with it!
